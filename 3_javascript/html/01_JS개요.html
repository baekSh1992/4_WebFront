스크립트(Script) 언어

        자바와는 연관성이 하나도 없다. (성만 같고 다른 이름)
        자바 = 코드 -> 컴파일 -> 실행

        스크립트 언어
        = 기본 프로그램의 동작을 사용자의 요구에 맞게 수행되도록 해주는 용도의 언어

        별도의 컴파일을 진행하지 않고 
        인터프리터(interpreter)를 이용하여 소스코드를 한 줄씩 읽어 바로 실행
        (직독직해 = 한 줄 읽고, 한 줄 해석...반복)
        -> 때문에 컴파일에 소요되는 시간이 없음.
           단, 프로그램 실행 시간이 느리다. (그때 그때 읽어서 해석하기 때문)

Javascript 란?

        웹 브라우저에서 많이 사용하는 인터프리터 방식의 객체 지향 프로그래밍 언어

        ECMA Script 표준을 따르는 대표적인 웹 기술이다.

자바스크립트 코드 작성법(4가지)

  • 브라우저 콘솔에 직접 작성(웹에서 개발자모드 > 콘솔)
  • 태그에 직접 JS 코드를 작성(inline방식)
  • html 내부에 script 태그를 이용하여 작성(internal)
  • [★] html 외부에 파일(.js)을 이용해서 작성(external)
