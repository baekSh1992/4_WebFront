요소의 영역(여백) 관련 속성

        모든 html 요소는 총 4가지의 영역으로 구성되어 있다.

        1. content 
            - 요소의 내용이 작성되어지는 영역
            (시작, 종료태그 사이에 작성되는 내용이 출력되는 영역)

            
        2. padding : 
            - content와 border 사이의 영역 

        3. border(테두리) : 
            - 요소의 테두리 영역
            - content 보다 바깥쪽에서 감싸고 있음.

        4. margin(여백,차이) 
            - 다른 요소와의 간격

box-sizing

        요소의 전체 크기가 가로 세로 300px인 정사각형

        * box-sizing 속성
        - width / height 속성이 적용되는 범위를 지정하는 속성

        1) content-box (기본값) : width / height 적용 범위를 content로 한정

        2) box-sizing: border-box; -> border * 2 + padding * 2 + content(x) = border-box 자동계산
            width / height 적용 범위를
            border + padding + content의 크기의 합과 동일하게 계산해서
            content의 크기를 자동 지정
